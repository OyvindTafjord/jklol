jklol

jklol is a Java library for developing structured machine learning models. jklol supports linear classification, structured prediction, graphical models, likelihood-based and max-margin methods, semi-supervised and weakly-supervised learning, generalized matrix / tensor factorization, and neural networks. It is designed to be intuitive, fast, and modular, with the main goal of reducing development time for new machine learning models.

Why use jklol ...

... at all? Because it will save you time. Avoid reimplementing basic algorithmic building blocks. (How many times have you implemented gradient descent?)

... as opposed to something else? Because it is more flexible. When developing a machine learning model, especially for research purposes, you need the ability to change your system in various ways. Maybe you want to use unlabeled data, or try out max-margin learning, or add a weird constraint to your objective. jklol's modular design allows you to independently swap out inference algorithms, model structures, parametrizations, objectives and optimization routines, making it easy to tinker.

Features

jklol supports:
  • Training and inference for general graphical models. Both loglinear and max-margin training are supported, as are graphical models with repeated structure (e.g., sequence models).
  • Semi-supervised and weakly-supervised training, using expectation maximization or stochastic gradient.
  • Other structured prediction. (e.g., parsing)
  • A tensor library for fast mathematical computation.
  • Fast optimization via (stochastic) gradient descent or L-BFGS. All optimization algorithms exploit parallelization for reduced training times.

Sample Applications

I've used jklol for:

  • Part-of-speech tagging and CCG supertagging
  • Broad coverage CCG parsing and semantic parsing
  • Training models of vector space composition
  • Miscellaneous miniature graphical models
  • Matrix factorization ala universal schema
(Basically, the experiments for anything I've published in the past 2 years use jklol.)
